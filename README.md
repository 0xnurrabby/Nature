## ✦ Features

<div align="center">

| | Feature | What it does |
|:---:|---|---|
| 📸 | Photo gallery | Showcases nature photography of untouched places |
| ✍️ | Nature writings | Thoughtful reflections and observations about the natural world |
| 🎬 | Background video | Atmospheric video background for an immersive feel |
| 🌿 | Calm aesthetic | Designed to feel peaceful, not busy |
| ♻️ | Environmental message | Gentle reminders about protecting nature |

</div>

## ✦ Download & Run

**Step 1** .... Clone the repo

```bash
git clone https://github.com/0xnurrabby/Nature
cd Nature
```

**Step 2** .... Open in browser

```bash
start index.html
# No build or dependencies needed
```

**Step 3** .... Browse the gallery

---

## ✦ Setup

```
1. Clone the repo
2. Open index.html in any browser
```
